# Autonomous-GPS-Vehicle

## Project Description

## System Architecture
  ### Inputs
  Camera (For the Computer Vision aspect)
  LiDAR Sensor (For detecting objects)
  Magnetometer (For orientation in areas with poor GPS signal)
  
  ### Outputs
  Motor controls (i.e. wheels)
  Speaker (Audio; for verbally communicating what object it sees)
  
  ### Hardware Components
  Jetson Nano
  LiDAR Sensor
  
  
  ### Software Components
  Robot Operating Sensor (ROS)
  #### Libraries
  OpenCV
  IMU/Robot_Localization
  RPLiDAR_ROS
